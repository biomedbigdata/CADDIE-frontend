Cancer Driver Drug Interaction Explorer


The Cancer Driver Drug Interaction Explorer (CADDIE) is a systems medicine platform which uses network biology algorithms for drug repurposing in cancer. A variety of different data resources have been implemented for cancer driver gene information as well as interaction data for gene-gene interactions and gene-drug interactions. We further integrated genetical background data like related diseases and mutation data to improve the algorithms and to facilitate the interpretation of the results.
We provide a user friendly graphical interface which allows to inspect and to understand the biological context of the predictions. CADDIE can be used to repurpose drugs as well as to find new drug targets.

General workflow of the platform

  1. Choose which source should be used for cancer driver genes, gene-gene interactions as well as which cancer type should be highlighted.
  2. Select seed nodes in the network. The algorithms will find putative drug-targets or drugs based on these seed nodes.
  3. Run the analysis on our server and inspect the results as soon as it is done. A network visualization helps to interpret the results and all data can be downloaded for further processing.

Data Privacy

This website only stores data to provide full functionality to the user. Upon starting an analysis task in the explorer, a task identification string is anonymously stored in your browser. This enables you to revisit previous analysis tasks without storing the URLs and to select tasks to summarize them. When removing the data of this website, previous analysis tasks will not appear in the task list anymore.
This website uses no cookies or tracks of user data in any way.
