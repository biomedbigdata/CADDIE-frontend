How to use CADDIE

CADDIE (Cancer Driver Drug Interaction Explorer) is a resource to repurpose drugs in cancer based on gene-gene and drug-gene interaction networks. The user has the option to choose between the implemented interaction networks from STRING, BioGRID, DrugBank, ChEMBL and more. CADDIE then exploits the information in selected networks using network algorithms like Trustrank or Degree Centrality. The algorithms and the desired parameters are again customizable by the user. Doing this, CADDIE is able to identify drug repurposing candidates or to search for putative drug targets.

Workflow

The main steps of working with CADDIE are:

  1. Pick your cancer type of interest
  2. Select the gene-interaction database of your choice
  3. Find seed genes
  4. Start a drug-target or drug search using one of the implemented algorithms
  5. Visualize the results and download them for further processing

Variations of this workflow are of course possible, for example you have the option to upload your own list of seed genes or to find seed genes based on related diseases. In general, CADDIE facilitates the seed selection process with a variety of different types of implemented information.

Seed genes

Seed genes are the starting genes for the algorithms. All implemented algorithms search in the interaction network for drug-targets or drugs based on their relation the the seed genes. Therefore, the seed genes are your genes of interest for which putative drugs or drug-target candidates should be found for.

Algorithms

CADDIE comes with algorithms for both drug-target identification and drug candidate search. Drug-target algorithms give you the possibility to find hub genes in the human gene-interaction network that are central to your seed genes. They also can be a helpful preprocessing step for your drug search. We implemented 6 different drug-target search algorithms in CADDIE:

  • Mutli-Steiner Tree: The Multi-level Steiner tree algorithm is used to approximate the minimum spanning tree spanned by the seed nodes in reasonable time. The implementation is adopted from Ahmed et al. It can be used to create a minimum spanning subnetwork between user-selected seed nodes, which happen to be central interaction partners between the seed nodes, and thus represent favorable drug-targets.
  • KeyPathwayMiner: KeyPathwayMiner (KPM) is an online tool developed by Alcaraz et al. for pathway enrichment analysis. Users have the option to utilize KPM for their drug-target search by selecting seed nodes in the GGI network and letting KPM find an interaction network of genes that are commonly dysregulated spanned by the seed genes. The resulting proteins are presumably functionally related to the seed nodes, and therefore are suitable drug-target candidates. Only one parameter K has to be set by the user which defines the amount of permitted intermediate nodes that are neither part of the seed nodes nor the common pathway.
  • TrustRank: TrustRank is based on the same concepts as the Google PageRank algorithm and Closeness Centrality. A crawler searches the network based on user-selected seeds and ranks visited nodes, damping the score based on the distance traveled. The damping factor can be set by the user in a range from 0-1, with a higher damping factor causing the crawler to explore nodes in close proximity or in larger portions of the GGI network. In CADDIE, TrustRank is able to find putative drug targets as well as drug candidates.
  • Degree Centrality: Degree centrality measurement is obtained by ranking the nodes in a network based on their degree, which is defined as the number of neighbors a node has divided by the number of nodes. Its normalized form, which is implemented in CADDIE, can be described as
    centralitydegree(x) = deg(x) / n − 1
    where x is a given node and deg(x) is its degree. While it is a commonly used network analysis technique, it most importantly has been shown useful in the identification of essential proteins in PPI networks. Thus, it is a simple approach to classifying the network-related importance of a particular protein. In CADDIE it can be used to discover valuable drug-targets or drugs, based on the seed selection given by the user.
  • Harmonic Centrality: Harmonic centrality measurement can be described as the average shortest distance from each node to all other nodes in a network. This measurement is the equivalent of closeness centrality for disconnected graphs. For comparable results, the normalized version was used. Formally speaking, it can be annotated as
    centralityharmonic(x) = ∑y≠xy( (n − 1) / dist(x, y) )
    where x is a given node and n − 1/dist(x, y) = 0 if dist(x, y) = ∞ (reference). The closer a node is to other nodes, the higher the score. It has already been proven successful in a number of biological network problems for instance with metabolic or PPI networks . In CADDIE, this algorithm can be used to find drug targets or drugs that affect the seed nodes as direct as possible.
  • Betweenness Centrality: Betweenness is obtained by finding the shortest paths for each pair of nodes in the network and assessing the number of shortest paths that pass through a particular node, such that a measure of the centrality of a node in a network global context is received. Betweenness Centrality has been established as a common measurement in network biological application and is especially practical in finding communities in large networks. In CADDIE, betweenness is based on the shortest paths between the seed nodes only and can be used to find drug targets with maximized connectivity to all seeds.

Furthermore, we integrated 4 drug candidate search algorithms. These tasks will return a list of drugs and their relations to your seed genes.

  • TrustRank: See description above.
  • Harmonic Centrality: See description above.
  • Degree Centrality: See description above.
  • Network Proximity: Network Proximity, as introduced by Guney et al., is the average length of shortest paths from drug target nodes to all of the user-selected seed nodes. The algorithm then computes a statistical significance score comparing against random expectation. This algorithm was adopted in CADDIE so that best-scored drugs are returned to the user as candidate drugs.

All of the algorithms can be adjusted to your needs with parameters. We also provide custom options such as filtering out unapproved or nutraceutical drugs.

Explorer

The Explorer is the main part of the application. It allows you to explore the interactome of you cancer type or cancer driver genes of interest. Most importantly, it gives you a variety of information like comorbidities or expression values to help you select seed genes for your drug or drug-target search.

Left Menu

On the left side you can find a menu bar which gives you the options to select your cancer type or gene interaction resource of interest. You will receive a short overview over your cancer type of interest. After selecting your seed genes, the left menu allows you to start analysis tasks (max. 3 at a time). The tasks will be displayed at the bottom of this menu in the 'Tasks' section. A click on a finished task leads you to the analysis view.

Gene-Interaction Network

On the right side of the explorer is the gene-interaction network. It contains all cancer driver genes of your selected cancer type, forming a minimal spanning tree. It is supplemented by genes found by the multiple steiner tree algorithm in order to make the minimal spanning tree possible. The network gives you an overview over the relations of the cancer driver genes to each other and lets you select seed genes for the drug-target or drug search by double-clicking the nodes. A single click on a node opens a window at the top of the left menu with detail information about the drug or gene such as comorbidities or the node degree in the complete human drug-gene interaction network.

Below the network you can find various functions that help you to manipulate the network to find genes of your interest. The expression of genes in different tissues (data from GTEx) as well as in different cancer types (data from TCGA) can be highlighted and genes may be selected automatically above a certain threshold. The same possibility exists for mutation data from TCGA. Furthermore, a menu is available with versatile options such as selection of all cancer driver genes or upload of a vcf-file to filter for genes with damaging mutations.
You can add your own genes to the network by uploading your own seed genes (for example via the 'Custom genes' function in the left menu) followed by the 'Add to network' functionality. Gene-gene interactions are taken from the selected interaction database.

Analysis View

THe analysis view opens when you click on a finished task in the left menu in the explorer. The analysis view will open on top of the network, and is divided in four parts. One part is the result network, which offers the same functions as the initial gene-network but here it displays all seed genes in combination with the found genes and/or drugs and allows you to inspect the relations from the seeds to the results. Your can also look at your results in a table format, where additional information to each of the genes or drugs is displayed. Here you can sort the nodes to get a better overview and download the tables for further processing on your side. Furthermore, CADDIE also provides different statistics and graphs about the analysis (depending on your analysis parameters). For example, you can set the calculated score for each of your genes and drugs in comparison to the node degrees in the drug-gene interactome (see below). Ultimately, the selected parameters and versions of the used datasets can be retrieved in the fourth tab.

Drug Interaction Lookup

The drug interaction lookup enables you to inspect the interaction datasets in respect to individual drugs. If the searched drug is not found in the database, the drug with the closest matching name will be shown.
All interacting genes based on the selected drug-gene interaction dataset will be listed. Cancer driver genes for any cancer type in the CADDIE database will be highlighted.

Gene Interaction Lookup

The gene interaction lookup works similar to the drug interaction lookup with the difference that that it allows you to search for genes instead of drugs. The list of drug interactions will be shown along with detailed drug information. In addition, you will receive a list of all known cancer types related to your gene of interest.
